## ✅ On-Page SEO Checklist

- [ ] `<title>` includes primary keyword + brand name
- [ ] `<meta description>` is 150–160 characters, includes CTA
- [ ] `<h1>` is unique per page and contains target keyword
- [ ] Image `alt` text includes descriptive keywords
- [ ] All internal links use descriptive anchor text
- [ ] Page loads in under 1.5 seconds (use Lighthouse)
- [ ] Schema JSON-LD added to `<head>`
- [ ] Open Graph meta tags added for social sharing
- [ ] Sitemap.xml created and submitted to Google Search Console
- [ ] robots.txt configured correctly
- [ ] Google Analytics / Search Console connected
- [ ] SSL/HTTPS enabled on deployment
